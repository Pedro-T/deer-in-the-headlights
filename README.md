# Deer in the Headlights

A short game about a deer trapped on the freeway trying to get out. Work in progress as an entry to the 2025 20 Second Game Jam.

## How To Play
- Move the deer using WASD or the arrow keys
- Your goal is to reach the exit point at the top of the play area
- You lose if the deer is hit by any vehicle.
- There is a twenty second timer. If it expires before you reach the exit, you lose!
- The exit is not available from the start!
